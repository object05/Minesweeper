How to start a music:
1.Tools -> 8bitSoundGenerator -> MusicEditor
2.Drag your mouse to add melody
	shift & drag can select
	ctrl+c copy selected
	ctrl+x cut selected
	ctrl+v paste at end
	delete remove selected
3.save music to a file
4.add class Music to load a muisc
5.run Music.Play

How to start a sound effect:
1.new a SoundClip
2.Contruct SoundClip from xxHz to xxxHz
	Min hz is 10
	Max hz is 800
3.SoundClip.Data and set data to a AudioClip
4.Use AudioClip as normal
